# DRAM Trace Samples

Small `dram_trace.csv` files placed here can be used for quick DRAM power
simulation without running a full mixed simulation first.

## Format

```
cycle,kind,addr,id,size_bytes,beats,bytes,clock_hz,mem_base
```

Generated by `2_POSTSIM/glsim/run_mixed.sh` with `DUMP_DRAM_TRACE=1`.

## Usage

```sh
cd /home/sjm/test/tapeout/3_POWER
DRAM_TRACE_FILE=dramtrace/dram_trace.csv ./run_dram_power
```
