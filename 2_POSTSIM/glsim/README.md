# Mixed Gate-Level Simulation

`glsim/` builds and runs a VCS simulation that combines the synthesized
`BuckyballAccelerator` gate-level netlist with the surrounding RTL testbench.
It can emit VCD/FST waveforms and SAIF activity for PTPX.

## Files and Subdirectories

- `compile_mixed.sh`: generates the mixed filelist, patches queue RAMs, extracts
  the standard-cell Verilog model, and builds `build/simv_mixed`.
- `run_mixed.sh`: runs `build/simv_mixed` with an ELF workload and waveform/SAIF
  controls.
- `tb_glsim_mixed.sv`: mixed-simulation testbench.
- `vc_hdrs.h`: VCS header support.
- `scripts/`: helper scripts for filelist generation, netlist patching, and VCD
  filtering.
- `elf/`: default and user-provided ELF workloads.
- `build/`: generated compile products.
- `results/`: generated simulation runs.
- `csrc/`: VCS C/C++ support area. It includes `vcs_bbsim_dpi.cpp`; most other
  files are generated by VCS.

## Compile

```sh
cd /home/sjm/tapeout/2_POSTSIM/glsim
./compile_mixed.sh
```

Common environment overrides:

- `NETLIST`: synthesized accelerator netlist. Default:
  `3_POWER/netlist/BuckyballAccelerator.v`.
- `BUILD_DIR`: generated build directory. Default: `glsim/build`.
- `SIMV`: simulator output path. Default: `build/simv_mixed`.
- `RTL_DIR`: RTL source directory. Default: `0_RTL/RTL`.
- `SRAM_ROOT`: real SRAM compiler-run directory. Default:
  `0_RTL/real_sram_libs/compiler_runs`.
- `FALLBACK_BUILD`: optional fallback directory for missing RTL filelist entries.
- `RANDOMIZE_RTL`, `ZERO_RANDOM`, `INITREG_SUPPORT`: VCS random/initreg control.

## Run

Default root-flow equivalent:

```sh
cd /home/sjm/tapeout/2_POSTSIM/glsim
ELF=/home/sjm/tapeout/2_POSTSIM/glsim/elf/default.elf \
CYCLES=5000000 \
PROGRESS=200000 \
DUMP_VCD=1 \
BB_SAIF=1 \
./run_mixed.sh
```

Common environment overrides:

- `SIMV`: simulator executable. Default: `build/simv_mixed`.
- `ELF`: workload ELF.
- `CYCLES`: cycle limit. Default: `5000000`.
- `PROGRESS`: progress interval. Default: `100000` in this script and `200000`
  when launched by `run_all`.
- `RUN_NAME`: result directory name. Default: timestamped.
- `OUT_DIR`: result directory. Default: `results/$RUN_NAME`.
- `TIMINGCHECKS`: `0` disables timing checks.
- `DUMP_VCD`, `DUMP_FST`, `DUMP_SAIF`: waveform/activity controls.
- `BB_SAIF`: enables the Buckyball-specific SAIF window.
- `VCD_FILTER`: filters accelerator internal scopes from the final VCD.
- `ZERO_INIT`: adds `+vcs+initreg+0`.
- `EXTRA_ARGS`: extra simulator plusargs/options.

Generated files usually include:

- `results/$RUN_NAME/glsim_mixed.saif`
- `results/$RUN_NAME/glsim_mixed.vcd`
- `results/$RUN_NAME/glsim_mixed.fst`
- `results/$RUN_NAME/stdout.log`
- `results/$RUN_NAME/sim.log`
